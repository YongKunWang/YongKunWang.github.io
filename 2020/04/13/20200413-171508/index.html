import argparse

使用过程3：因为 argparse 会把我们传递给它的选项视作为字符串 ,增加类型信息

import argparse

# 得到终端的参数信息
args = parser.parse_args()
# 通过args.(add_argument增加的参数)可直接进行显示
print(args.echo**2)

"""
python demo1.py 2

前馈网络的简化写法

每一层的输出直接作为下一层的输入，这种网络称之为前馈神经网络feedforward neural network。

对于此类的网络如果每次都写复杂的forward函数会特别麻烦，再次就有两种简化方法：

  • nn.ModuleList()：
  • nn.Sequential()：这是一个特殊的module，它可以包含几个子module,可以像list一样使用他，但不能直接把输入传入到ModuleList

nn.Sequential()对象

nn.Sequential()对象是类似keras的前馈模型的对象，可以为之添加层实现前馈神经网络。

"""
nn.Sequential()的用法
"""

import  torch
import  torchvision
import torch.nn as nn
import torch.nn.functional as F
# 第一种方法
net1 = nn.Sequential()
net1.add_module('conv1', nn.Conv2d(3,3,3))
net1.add_module('batchnorm', nn.BatchNorm2d(3))
net1.add_module('activate_layer',nn.ReLU())
print(net1)

"""
Sequential(
  (conv1): Conv2d(3, 3, kernel_size=(3, 3), stride=(1, 1))
  (batchnorm): BatchNorm2d(3, eps=1e-05, momentum=0.1, affine=True, track_running_stats=True)
  (activate_layer): ReLU()
)
"""
# 第二种方法
net2 = nn.Sequential(
    nn.Conv2d(3,3,3),
    nn.BatchNorm2d(3),
    nn.ReLU()
)
print(net2)
"""
Sequential(
  (0): Conv2d(3, 3, kernel_size=(3, 3), stride=(1, 1))
  (1): BatchNorm2d(3, eps=1e-05, momentum=0.1, affine=True, track_running_stats=True)
  (2): ReLU()
)
"""
# 第三种方法
from collections import OrderedDict

net3 = nn.Sequential(
    OrderedDict(
        [
            ('conv',nn.Conv2d(3,3,3)),
            ('batchnorm', nn.BatchNorm2d(3)),
            ('activate',nn.ReLU())
        ]
    )
)
# print 对象即可
print(net3)
"""
Sequential(
  (conv): Conv2d(3, 3, kernel_size=(3, 3), stride=(1, 1))
  (batchnorm): BatchNorm2d(3, eps=1e-05, momentum=0.1, affine=True, track_running_stats=True)
  (activate): ReLU()
)
"""

# 提取Module对象
# 可以根据名字或者序号取出子module
print(net1.conv1)
print(net2[0])
print(net3.conv)
"""
Conv2d(3, 3, kernel_size=(3, 3), stride=(1, 1))
Conv2d(3, 3, kernel_size=(3, 3), stride=(1, 1))
Conv2d(3, 3, kernel_size=(3, 3), stride=(1, 1))
"""

# 调用模型
print('----------输入参数--------')
input = torch.randn(1,3,4,4)
print(input)

print('----------输出参数--------')
output = net1(input)
print(output)

nn.ModuleList()对象

ModuleList是Module的子类，当在Module中使用它的时候，就能自动识别为子module。

和普通list不一样，它和torch的其他机制结合紧密，继承了nn.Module的网络模型class可以使用nn.ModuleList并识别其中的parameters，当然这只是个list，不会自动实现forward方法， 

"""
nn.ModuleList()对象
"""
import  torch
import  torchvision
import torch.nn as nn
import torch.nn.functional as F

modelList = nn.ModuleList(
    [
        nn.Linear(3,4),
        nn.ReLU(),
        nn.Linear(4,2)
    ]
)

input = torch.randn(1,3)
for model in modelList:
    input = model(input)
print(input)

## error,因为modelList未实现forward方法
#output = modelList(input)
class MyModel(nn.Module):
    def __init__(self):
        super(MyModel, self).__init__()
        ## 普通的list
        self.list = [nn.Linear(3,4), nn.ReLU()]
        ## ModuleList
        self.modules_list = nn.ModuleList([nn.Conv2d(3,3,3), nn.ReLU()])
    def forward(self):
        pass
model = MyModel()
print(model)
"""
MyModel(
  (modules_list): ModuleList(
    (0): Conv2d(3, 3, kernel_size=(3, 3), stride=(1, 1))
    (1): ReLU()
  )
)
"""
for name, param in model.named_parameters():
    print(name," " ,param.size())
"""
modules_list.0.weight   torch.Size([3, 3, 3, 3])
modules_list.0.bias   torch.Size([3])
"""

可见，list的字module并不能被主module所识别，而ModuleList中的字module能够被主module所识别！

这意味着如果用list保存子module，将无法调整其参数，因其未加入到主module的参数中。

除ModuleList之外还有ParameterList，其是一个可以包含多个parameter的类list对象。在实际应用中，使用方式与ModuleList类似。如果在构造函数__init__中用到list、tuple、dict等对象时，一定要思考是否应该用ModuleList或ParameterList代替。

关于bath normalization

有偏置b的卷积层不适合和batch normalization层一起出现

偏置b在网络中的目的是使得当某一层layer的输入分布不是0为中心时给予偏置补偿（比如简单来看，要拟合y=wx+b，b=100， 那么用没有偏置的单层网络y=wx是怎么都无法拟合的！）

但是，当该层layer的输入经过归一化，也就是0为中心时，不要偏置也可以拟合（对应上述的例子，就是把所有数据移到原点附近，这时候y=wx就能拟合，拟合方向就行）
