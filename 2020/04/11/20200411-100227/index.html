单向链表结构

实际内存结构

  • 链表是以节点的方法来存储的，属于链式结构
  • 每一个节点包含data域和next域，且next域指向下一个节点
  • 链表的各个节点不一定是连续存储的
  • 链表分为由头节点的和没有头结点的链表，需要根据实际需求进行确定
  • 单链表的逻辑结构示意图：

单链表的应用实例

场景：

客户端将某人的在线用户返回给后端，后端不能操作数据库，直接在内存中进行操作，将排好序的在线用户返回给客户端，以供使用！

单链表基本操作

单链表分析

所需要的变量定义：

  • 创建一个节点类Node
    • data域 private
    • next 域 public，赋初值为next = null
    • 构造函数，接收data域的数据,无需接收next域的数据
  • 创建链表管理类：
    • 需要一个头结点对象
    • 在进行遍历操作是都需要使用一个指针变量(Node)来标记本次循环的位置信息
      • 增删改查的实现中都会用到
    • 判断到了链表的尾部：
      • temp.next== null
    • 判断链表是否为空：
      • headNode.next == null
    • 因为链表的结构，无需判断链表是否满

思路分析

  • 单链表的尾部创建（从头开始创建）：

    • 创建节点类Node，表示相应的节点信息

    • 创建一个链表管理类

    • 链表管理类创建头结点，表示单链表的头部，此节点信息不可动

    • 新建辅助节点temp进行遍历操作，因为头结点不可动

    • 此temp相当于指针，记录当前处于整个链表的什么位置

    • 进行节点遍历，判定最后节点的条件：Node.next = null

    • 将最后一个节点的Node.next = newNode

  • 显示链表信息

    • 判断链表是否为空
    • 创建临时变量指向头结点后的第一个元素
    • 判断此节点next域是否为null
    • 打印节点信息
    • 临时变量向后指

注意：这里的临时变量的定义有些区别！！！

代码实现：

class LinkedList{
    //新建头结点
    private UserNode headNode = new UserNode(0, "");
    //添加节点操作
    public void add(UserNode node) {
        //创建指针,指向头部
        UserNode temp = headNode;

        while(true) {
            //当指向链表尾部时的判定条件
            if(temp.next == null)
                break;
            //向前一步
            temp = temp.next;
        }
        //直接链接在最后
        temp.next = node;

    }
    //显示链表节点信息
    public void show() {
        //判断链表是否为空
        if(headNode.next == null) {
            System.out.println("链表为空~~");
            return ;
        }
        UserNode temp = headNode.next;
        while(true) {
            if(temp == null) 
                break;
            System.out.println(temp);
            temp = temp.next;

        }
    }
}

class  UserNode {
    // data 域
    private int id;
    private String name;
    //next域
    public UserNode next = null;

    public UserNode(int id, String name) {
        this.id = id;
        this.name = name;
    }

    @Override
    public String toString() {
        return "UserNode [id=" + id + ", name=" + name + "]";
    }
}

  • 单链表按顺序创建：当id一致时要有提示信息

    • 创建临时变量temp指向头结点
    • 进行循环遍历到最后
    • new Node的data域和每一个temp的data域进行比较
    • 当符合条件时：（顺序不可改变！！）
      • new node.next = temp.next
      • temp.next = new node

代码实现：

public void addById(UserNode node) {
        UserNode temp = headNode;
        //判断是否存在相同的id
        boolean flag = false;
        while(true) {
            //插在最后面
            if(temp.next == null)
                break;
            //按顺序插入
            if(temp.next.getId() > node.getId()) {
                break;
            // 存在插入冲突
            }else if (temp.next.getId() == node.getId()) {
                flag = true;
                break;
            }
            temp = temp.next;
        }
        if(flag) {
            System.out.printf("%d 已经插入到链表中~~",node.getId());
        }else{
            node.next = temp.next;
            temp.next = node;
        }
    }
  • 单链表的修改操作：
    • 需要输入待修改的id
    • 定义temp作为指针，定义flag作为是否找到此id
    • temp向后进行遍历，找到以后置flag = true，跳出
    • 在跳出循环后进行判断和进行修改操作

代码实现：

/**
     * 1. 根据id找见要修改的节点
     * 2. 有两个条件退出循环，使用flag来判断到底是哪个条件退出循环
     * @param node
     */
    public void updateById(UserNode node) {
        if(headNode.next == null) {
            System.out.println("链表为空~");
            return;
        }
        UserNode temp = headNode;
        boolean flag = false;
        while(true) {
            if(temp.next == null) {
                break;
            }
            if(temp.next.getId() == node.getId()) {
                flag = true;
                break;
            }
            temp = temp.next;
        }
        if(flag) {
            temp.next.setName(node.getName());;
        }else {
            System.out.printf("未找到%s的用户\n",node.getId());
        }
    }

  • 单链表的删除操作：

    • 需要先找到 要删除节点的前一个节点temp
    • 进行操作：temp.next = temp.next.next
    • 被删除的节点将不会有其他的引用，会被垃圾回收机制回收

package com.homewyk.linked;

import java.util.Scanner;

public class LinkedListDemo {
    public static void main(String[] argc) {
        LinkedList link = new LinkedList();
        char key = ' ';
        Scanner scanner = new Scanner(System.in);
        boolean loop = true;
        System.out.println("欢迎使用单链表测试程序");
        while(loop) {
            key = scanner.next().charAt(0);
            switch (key) {
            case 'a':
                link.add(new UserNode(1, "w1"));
                link.add(new UserNode(2, "w2"));
                link.add(new UserNode(3, "w3"));
                link.add(new UserNode(4, "w4"));
                link.add(new UserNode(5, "w5"));
                System.out.println("在链表尾部添加节点完成");
                break;
            case 'i':
                link.addById(new UserNode(3, "w3"));
                link.addById(new UserNode(4, "w4"));
                link.addById(new UserNode(5, "w5"));
                link.addById(new UserNode(1, "w1"));
                link.addById(new UserNode(2, "w2"));
                System.out.println("按照顺序插入链表节点完成");
                break;
            case 'u':
                link.updateById(new UserNode(1, "ww1"));
                break;
            case 'd':
                System.out.println("请输入要删除的id");
                int id = scanner.nextInt();
                link.delById(id);
                break;
            case 's':
                link.show();
                break;
            case 'e':
                loop = false;
                break;
            default:
                break;
            }
        }
        System.out.println("退出程序~~");
    }
}
