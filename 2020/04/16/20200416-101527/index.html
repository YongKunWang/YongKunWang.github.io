栈(Stack)

  1. 栈的英文为(stack)

  2. 栈是一个先入后出(FILO-First In Last Out)的有序列表。

  3. 栈(stack)是限制线性表中元素的插入和删除只能在线性表的同一端进行的一种特殊线性表。允许插入和删除的一端，为变化的一端，称为栈顶(Top)，另一端为固定的一端，称为栈底(Bottom)。

  4. 根据栈的定义可知，最先放入栈中元素在栈底，最后放入的元素在栈顶，而删除元素刚好相反，最后放入的元素最先删除，最先放入的元素最后删除

  5. 出栈和入栈的概念

    始终指向栈顶元素

栈的应用场景

  1. 子程序的调用：在跳往子程序前，会先将下个指令的地址存到堆栈中，直到子程序执行完后再将地址取出，以回到原来的程序中。
  2. 处理递归调用：和子程序的调用类似，只是除了储存下一个指令的地址外，也将参数、区域变量等数据存入堆栈中。
  3. 表达式的转换[中缀表达式转后缀表达式]与求值(实际解决)。
  4. 二叉树的遍历。
  5. 图形的深度优先(depth一first)搜索法。

数组模拟栈

思路：

  1. 定义1个变量指向栈顶：top
  2. 赋初值top = -1
  3. 元素入栈操作：top ++; stack[top] = xx
  4. 元素出栈操作：数据类型 变量 = stack[top]; top --; return 变量

判断stack是否空|满

  • 满就是数组满了top = maxsize -1
  • 空就是top == -1
/**
     * 判断stack is full?
     * @return
     */
    public boolean isFull() {
        return top == maxsize - 1;
    }
    /**
     * 判读 stack is empty?
     * @return
     */
    public boolean isEmpty() {
        return top == -1;
    }

入栈操作

也容易记忆，因为top的初值为-1 ，要入栈，肯定要先自加，在入栈

  1. top ++
  2. stack[top] = num
/**
*  1.入栈
* @param num
*/
public void push(int num) {
    if (isFull()) {
        System.out.println("stack 已经满了~~");
        return;
    }
    top ++;
    stack[top] = num;
}

出栈操作

  1. int num = stack[top]
  2. top --
  3. return num
/**
* 2. 出栈操作
* @return
*/
public int pop() {
    if (isEmpty()) {
        System.out.println("stack 已经空了~~");
        throw new RuntimeException("stack 已经空了~~");
    }
    int number = stack[top];
    top --; 
    return number;
}

详细代码实现

package com.homewyk.stack;

import java.util.Scanner;

public class ArrayStackDemo {
    public static void main(String[] args) {
        System.out.println("数组模拟栈的测试程序");
        ArrayStack stack = new ArrayStack(4);
        String key = "";
        Scanner scanner = new Scanner(System.in);
        boolean loop = true;
        while(loop) {
            System.out.println("push: 入栈操作~~");
            System.out.println("pop: 入栈操作~~");
            System.out.println("list: 显示元素~~");
            System.out.println("exit: 程序退出~~");
            key = scanner.next();
            switch (key) {
            case "push":
                System.out.println("入栈操作~~");
                System.out.println("请输入一个数：");
                int num = scanner.nextInt();
                stack.push(num);
                break;
            case "pop":
                System.out.println("出栈操作~~");
                System.out.println("pop:" + stack.pop());
                break;
            case "list":
                System.out.println("显示所有元素~~");
                stack.list();
                break;
            case "exit":
                loop = false;
                break;

            default:
                break;
            }
        }
        scanner.close();
        System.out.println("程序退出~~");
    }
}

class ArrayStack{

    private int maxsize;
    private int[] stack = null;
    private int top;
    /***
     * 构造函数
     * @param maxsize
     */
    public ArrayStack(int maxsize) {
        this.maxsize = maxsize;
        this.stack = new int[maxsize];
        top = -1;
    }
    /**
     * 判断stack is full?
     * @return
     */
    public boolean isFull() {
        return top == maxsize - 1;
    }
    /**
     * 判读 stack is empty?
     * @return
     */
    public boolean isEmpty() {
        return top == -1;
    }
    /**
     *  1.入栈
     * @param num
     */
    public void push(int num) {
        if (isFull()) {
            System.out.println("stack 已经满了~~");
            return;
        }
        top ++;
        stack[top] = num;
    }
    /**
     * 2. 出栈操作
     * @return
     */
    public int pop() {
        if (isEmpty()) {
            System.out.println("stack 已经空了~~");
            throw new RuntimeException("stack 已经空了~~");
        }
        int number = stack[top];
        top --; 
        return number;
    }
    /**
     * 列出所有的节点
     */
    public void list() {
        if (isEmpty()) {
            System.out.println("stack 已经空了~~");
            throw new RuntimeException("stack 已经空了~~");
        }
        for (int i = 0; i <= top; i++) {
            System.out.println("stack[" + i + "] : " + stack[i]);
        }
    }

}
