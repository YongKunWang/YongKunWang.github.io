总结广播语义

如果Pytorch操作支持广播,则其张量参数可以自动扩展为相同大小（不需要复制数据）。

一般语义

2个规则：

  • 每个张量至少有一个维度(这里表明标量不可以)
  • 在遍历维度是，从尾部（小维度）开始遍历
    • 两个Tensor必须满足一项条件：
      • 大小必须相等
        • a = torch.Tensor(2,3,4); b = torch.Tensor(2,3,4)
      • 他们其中一个要么是1，要么不存在
        • a = torch.Tnesor(2,3,4); b = torch.Tensor(3,1);

例子

x = torch.Tensor(2,3,5)
y = torch.Tensor(2,3,5)
print((x+y).shape)

x = torch.Tensor(2,3,5)
y = torch.Tensor(3,1)
print((x+y).shape)

x = torch.tensor(2.)
print(x.shape)
y = torch.Tensor(2,3,5)
print((x+y).shape)

x = torch.Tensor(2,3,5)
y = torch.Tensor(4,1)
print((x+y).shape)
"""
torch.Size([2, 3, 5])
torch.Size([2, 3, 5])
torch.Size([])
torch.Size([2, 3, 5])
RuntimeError: The size of tensor a (3) must match the size of tensor b (4) at non-singleton dimension 1

"""

如果x和y可以被广播,得到的张量大小的计算方法如下：

  • 如果维数x和y不相等，在维度较少的张量的维度前加上1使它们相等的长度。
  • 然后,对于每个维度的大小,生成维度的大小是x和y的最大值。
# 可以排列尾部维度,使阅读更容易
>>> x=torch.FloatTensor(5,1,4,1)
>>> y=torch.FloatTensor(  3,1,1)
>>> (x+y).size()
torch.Size([5, 3, 4, 1])

# 但不是必要的:
>>> x=torch.FloatTensor(1)
>>> y=torch.FloatTensor(3,1,7)
>>> (x+y).size()
torch.Size([3, 1, 7])

>>> x=torch.FloatTensor(5,2,4,1)
>>> y=torch.FloatTensor(3,1,1)
>>> (x+y).size()
RuntimeError: The size of tensor a (2) must match the size of tensor b (3) at non-singleton dimension 1
