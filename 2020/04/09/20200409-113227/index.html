环形队列结构

对数组模拟队列的不足点进行改造

将数组看作是一个环形的（可以通过取模的方法实现）

环形队列分析

所需要的变量定义：

  • 一个数组int[] arr
  • 最大容量：int[] arr = new int[maxsize]
  • 一个指针指向队列头部，（包含） int front = 0
  • 一个指针指向队列尾部的下一个 （不包含）int rear = 0

注意：

上述的两个指针的含义发生了很大的变化！！

发生变化的原因是通过使用队列尾部的下一个元素,空出一个空间出来，作为约定

环形思路分析

  1. 创建一个class，并实现上述变量的定义以及初始化

  2. 实现2个判断：

    • isFull -> （rear +1）% maxsize = front

      兼顾上述两种情况，统一使用取模来计算

    • isEmpty -> front == rear

      • 一个数据未添加时空front = 0, rear = 0; -> front == rear

      • 当取出数据空： rear = front

  3. 添加数据：

    • 判断是否满
    • 添加操作
    • 指针变化

  4. 取出数据：

    • 判断是否空
    • 取出数据
    • 指针变化

  5. 得到队列头部数据：

    • 判断是否为空
    • 使用front来获得队列头部数据

  6. 获得队列容量

    • (rear+maxsize -front)%maxsize

      我们一般计算容量都是结尾坐标减去开始坐标；

      • 比如下面的常规情况：
      • 针对于：rear在front之后的情况，其实可以直接计算：rear - front

      • 非常规情况：
      • 针对于：rear在front之前的情况，计算容量其实也是结束坐标减去开始坐标，但是这里的结尾坐标是加上一个容量，变为一个大数，最后在取余

package com.homewyk.queue;

import java.util.Scanner;

import org.junit.Test;

public class CircleQueue {

    @Test
    public void test() {
        MyCircleQueue queue = new MyCircleQueue(4);
        char key = ' ';
        Scanner in = new Scanner(System.in);
        boolean loop = true;
        while(loop) {
            System.out.println("s: 显示队列");
            System.out.println("e: 退出程序");
            System.out.println("a: 添加数据");
            System.out.println("g: 取出数据");
            System.out.println("h: 查看数据");
            //接收字符串的第一个字符
            key = in.next().charAt(0);
            switch (key) {
            case 's':
                queue.show();
                break;
            case 'e':
                in.close();
                loop = false;
                break;
            case 'a':
                int item = in.nextInt();
                queue.add(item);
                break;
            case 'g':
                queue.get();
                break;
            case 'h':
                System.out.println("队列头部元素为：" + queue.head());
                break;
            default:
                break;
            }
        }
        System.out.println("程序退出~~");
    }

}

class MyCircleQueue {
    private int[] arr = null;
    private int maxsize ;
    //注意此时的含义变了
    private int front;
    private int rear;

    /***
     * 构造函数，定义时确定队列容量
     * @param maxsize 队列容量
     */
    public MyCircleQueue(int maxsize) {
        this.maxsize = maxsize;
        this.arr = new int[maxsize];
        //指向当前元素
        front = 0;
        //指向队列的最后一个元素的后一个位置，留一个空位置
        rear = 0;
    }
    /***
     * 判断队列是否满
     * 3 x 1 2
     * front:2
     * rear:1
     * @return
     */
    public boolean isFull() {
        return (rear+1)%maxsize == front;
    }
    /***
     * 判断队列是否空
     * @return
     */
    public boolean isEmpty() {
        return rear == front;
    }
    /***
     * 添加队列
     * @param item
     */
    public void add(int item) {
        if(isFull()) {
            System.out.println("队列容量已满~~");
            return;
        }
        arr[rear] = item;
        rear = (rear+1)%maxsize;
    }
    /**
     * 取出队列元素
     * @return
     */
    public int get() {
        if(isEmpty()) {
            throw new RuntimeException("队列空，不能取数据");
        }
        int temp = arr[front];
        front = (front+1)%maxsize;
        return temp;
    }
    /***
     * 显示队列元素
     */
    public void show() {
        if (isEmpty()) {
            System.out.println("队列空，没有数据~~");
            return;
        }
        for (int i = front; i <front + this.size(); i++) {
            System.out.printf("arr[%d] = %d\n", i%maxsize, arr[i%maxsize]);
        }
    }
    public int size() {
        return (rear + maxsize - front) % maxsize;
    }
    /***
     * 显示队列头部，非取出
     * @return
     */
    public int head() {
        if (isEmpty()) {
            throw new RuntimeException("队列空，没有数据~~");
        }
        return arr[front];
    }
}
