中缀表达式

栈的实际需求：

请输入一个表达式

计算式:[722-5+1-5+3-3] 点击计算【如下图】

请问: 计算机底层是如何运算得到结果的？

注意不是简单的把算式列出运算,因为我们看这个算式 7 2 2 - 5,

但是计算机怎么理解这个算式的(对计算机而言，它接收到的就是一个字符串)，我们讨论的是这个问题。-> 栈

实现思路

简化思路：操作数为$[0,9]$,操作符为+-*/

  1. 建立一个index指针，来遍历我们的字符串表达式
  2. 如果我们发现是一个数字，直接入栈
  3. 如果扫描到是一个操作符：
    1. 如果当前栈为空，操作符入栈
    2. 若果当前栈非空，进行操作符优先级比较：
      1. 如果当前的操作符 <= 栈中的操作符，从数字栈中pop出2个数字，从操作符栈中pop出一个操作符，进行运算，然后将运算结果入数字栈，再将当前的操作符入栈
      2. 如果当前的操作符>栈中的操作符，直接入栈
  4. 当表达式扫描完毕后，就顺序进行数字栈和操作栈的出栈操作，并进行运行，第一个出栈的放在操作符后面，第二个出栈的放在操作符前面
  5. 到最后，数字栈中就剩下一个数字，就为表达式的结果

提高阶段

对于括号的处理：

可以这样想，在遇到形如 a * (b + c) 这样的形式的时候，左边的乘法是一定不能做的，我们只需要将左括号进栈即可。所以，我们可以把左括号看做是一个优先级无穷大的运算符，它使得左方的运算符都不能提前进行计算。

遇到右括号时，b + c 这个加法是可以进行运算的了，所以可以把右括号看作是一个优先级无穷小的运算符

它会使得操作符栈上的所有运算符都出栈并执行计算，直到遇到左括号。

遇到左括号以后，只需要把左括号从栈里弹出来，然后让它和右括号一起即可。

由于括号内的计算都已经完成了，结果是一个整数，我们已经在计算的过程中把这个整数放到操作数栈里了。所以整个括号内的求值就完成了。

思路

栈定义：

在出入栈和判断栈空满的基础上要添加新的方法：

  • 判断式数字还是操作符
  • 判断操作符的优先级（使用数字可以表示优先级的大小）
  • 进行一次运算操作
  • 获得栈顶元素
class ArrStack{

    private int maxsize;
    private int[] stack = null;
    private int top;
    /***
     * 构造函数
     * @param maxsize
     */
    public ArrStack(int maxsize) {
        this.maxsize = maxsize;
        this.stack = new int[maxsize];
        top = -1;
    }
    /**
     * 判断stack is full?
     * @return
     */
    public boolean isFull() {
        return top == maxsize - 1;
    }
    /**
     * 判读 stack is empty?
     * @return
     */
    public boolean isEmpty() {
        return top == -1;
    }
    /**
     *  1.入栈
     * @param num
     */
    public void push(int num) {
        if (isFull()) {
            System.out.println("stack 已经满了~~");
            return;
        }
        top ++;
        stack[top] = num;
    }
    /**
     * 2. 出栈操作
     * @return
     */
    public int pop() {
        if (isEmpty()) {
            System.out.println("stack 已经空了~~");
            throw new RuntimeException("stack 已经空了~~");
        }
        int number = stack[top];
        top --; 
        return number;
    }
    /**
     * 列出所有的节点
     */
    public void list() {
        if (isEmpty()) {
            System.out.println("stack 已经空了~~");
            throw new RuntimeException("stack 已经空了~~");
        }
        for (int i = 0; i <= top; i++) {
            System.out.println("stack[" + i + "] : " + stack[i]);
        }
    }
    /**
     * 判断字符是否为操作符
     * @param val
     * @return
     */
    public boolean isOper(char val) {
        return val == '+' || val == '-'|| val =='*' || val == '/';
    }
    /***
     * 运算
     * @param backwardPop 首次出栈值
     * @param forwardPop  第二次出栈值
     * @param oper
     * @return
     */
    public int calc(int backwardPop, int forwardPop, char oper) {
        int res = 0;
        switch (oper) {
        case '+':
            res = backwardPop + forwardPop;
            break;
        case '-':
            res = backwardPop - forwardPop;
            break;
        case '*':
            res = backwardPop * forwardPop;
            break;
        case '/':
            res = backwardPop / forwardPop;
            break;
        default:
            break;
        }
        return res;
    }
    /***
     * 得到操作符的优先级
     * @param oper
     * @return
     */
    public int priority(char oper) {
        int num = 0;
        switch (oper) {
        case '+':
            num = 0;
            break;
        case '-':
            num = 0;
            break;
        case '*':
            num = 1;
            break;
        case '/':
            num = 1;
            break;
        default:
            num = -1;
            break;
        }
        return num;
    }
    /**
     * 获得栈顶元素
     * @return
     */
    public int peek() {
        return this.stack[top];
    }
}

中缀表达式的实现：

根据上面的思路进行编写

class calculatr {
    private ArrStack numStack = new ArrStack(10);
    private ArrStack operStack = new ArrStack(10);
    private String expression;
    private int index = 0; 
    private boolean loop = true;
    private int backwardPop = 0;
    private int forwardPop = 0;
    private char oper = ' ';
    public calculatr(String expression){
        this.expression = expression;
    }

    public int CalcForMiddleExpression() {
        while(loop) {
            //beginIndex - the beginning index, inclusive.
            //endIndex - the ending index, exclusive.
            char ch = expression.substring(index, index+1).charAt(0);
            //是操作符，入操作栈
            if(operStack.isOper(ch)) {
                if (operStack.isEmpty()) {
                    operStack.push(ch);
                }else {
                    if (operStack.priority(ch) <= operStack.priority((char)operStack.peek())) {
                        forwardPop = numStack.pop();
                        backwardPop =  numStack.pop();
                        oper = (char) operStack.pop();
                        numStack.push(numStack.calc(backwardPop, forwardPop, oper));
                        operStack.push(ch);                        
                    }else {
                        operStack.push(ch);
                    }
                }
            }
            //不是操作符入数字栈
            else {
                numStack.push((ch - 48));
            }
            index ++;
            if(index == expression.length()) {
                loop = false;
            }
        }
//        System.out.println("num:");
//        numStack.list();
//        System.out.println("oper:");
//        operStack.list();

        loop = true;
        while(loop) {
            if (operStack.isEmpty()) {
                loop = false;
                break;
            }
            forwardPop = numStack.pop();
            backwardPop =  numStack.pop();
            oper = (char) operStack.pop();
            //System.out.println("calc:" + forwardPop + " : " + backwardPop + " : " + oper);
            int num = numStack.calc(backwardPop, forwardPop, oper);
            numStack.push(num);
        }
        return numStack.peek();
    }
}
